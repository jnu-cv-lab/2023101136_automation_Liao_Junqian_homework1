# 计算机视觉实验一：Python视觉开发环境搭建与图像基本读写

## 实验任务
1.  **任务1**：使用 OpenCV 读取一张测试图片
2.  **任务2**：输出图像基本信息，在终端打印：图像尺寸、图像通道数、像素数据类型（例如：宽度、长度）
3.  **任务3**：显示原图，用 Matplotlib 或 OpenCV 显示原图
4.  **任务4**：转换为灰度图，将彩色图转换为灰度图，并显示
5.  **任务5**：保存处理结果，把灰度图保存为新文件
6.  **任务6**：用 NumPy 做一个简单操作，例如输出某个像素值，或者把图像左上角一块区域裁剪出来保存
# 运行环境
- 操作系统：Ubuntu Linux
- Python 版本：3.12.3
- 依赖库：opencv-python、numpy、matplotlib
